eCacti

eCACTI is a tool used for finding an estimate of cache area, access times, and power dissipation for a given set of parameters (cache size, block size, associativity, and technology). [download page]

EXPRESSION

EXPRESSION is an Architecture Description Language (ADL) as well as a retargetable compiler / simulator tool-kit for architectural design space exploration (DSE). [download page]

TRIMARAN Patch

Patch for the Solaris version of the Trimaran EPIC compilation and performance monitoring system. This patch is for making Trimaran 2.0 (available for the HP-UX and Linux platforms) available for the Solaris 2.5 platform. (shell script)
