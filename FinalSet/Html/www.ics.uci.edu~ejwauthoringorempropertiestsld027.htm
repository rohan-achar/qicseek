DAV Links - Value

  • http://www.ietf.org/standards/dav/Link contains two elements src and dst

  • http://www.ietf.org/standards/dav/Src contains a URL pointing to the source of the link

  • http://www.ietf.org/standards/dav/Dst contains a URL pointing to the destination of the link
