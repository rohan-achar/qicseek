Distributed Authoring Group

Goal: to make distributed authoring as pervasive as browsing is today.

How?

  • By specifying:
    • Preliminary modifications to existing specifications (e.g., HTTP)
    • Usage conventions (e.g., URL name space)
    • Ensure that distributed web content authoring tools are broadly interoperable
