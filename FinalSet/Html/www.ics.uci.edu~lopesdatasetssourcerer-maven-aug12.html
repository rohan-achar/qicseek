Download page for Sourcerer Data Set: sourcerer-maven-aug12

"sourcerer-maven-aug12" is a tarball containing about 2,232 projects from the Maven Central Repository and stored in Sourcerer's repository format. Each project contains one or more versions. This data was collected in August 2012 using rsync from a mirror when Maven still allowed that. We are releasing this tarball so that this repository can be used as a reference collection for various research purposes. Release date: 11-14-2013

"sourcerer-maven-aug12" is a part of the UCI Source Code Data Sets.
