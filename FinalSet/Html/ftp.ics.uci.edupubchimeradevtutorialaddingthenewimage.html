Adding the New Image

  With the new image wright_flight.jpg downloaded. You can now create anchors on it and connect those anchors to Links. To do this follow the following actions.

Displaying the new image

Display the image by using the Load Image command in the File menu of Jimage.

Creating Anchors on the new Image

Create a couple of anchors on the image. Make an anchor around the aircraft and another around Orville.

Connecting the Anchors to the Active Link

Make sure that your are in bill_web and have selected places_to_go as the active link.

Put the mouse inside the anchor areas and click the right most mouse button. A pop-up menu will display some choices for you to select from. Choose the add to Active Link command for each anchor.

You have now made a Link which is connected to four anchors in three images. (places_to_go should now be connecting anchors in wright_flight.jpg, bill.gif, and earthrise.gif.) Now you can traverse the link from the various anchors.
