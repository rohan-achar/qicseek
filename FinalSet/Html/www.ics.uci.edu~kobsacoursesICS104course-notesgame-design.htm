Game design

Types of games:
action, shooter, adventure, role-playing, simulation, strategy, single vs. (massive) multi-player, etc.

Typical goals in "general HCI design":
reduce error rate, increase speed of task performance, improve retention, user satisfaction...

Additional predominant goal in game design: give users "fun"
  • challenges, rewards, fantasy, immersion, surprises, control/dominance, social interaction/bonding

Guidelines:
  • Game story
  • Game interface
  • Game mechanics
  • Game play

Non-entertainment applications: education, training, following regimens
