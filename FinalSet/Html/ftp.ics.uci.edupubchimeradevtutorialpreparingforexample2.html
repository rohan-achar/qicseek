Preparing for Creating an N-ary Link

  For the N-ary link example, please download the image wright_flight.jpg from the Chimera 2.0 website. This can be done by going to website:
Chimera 2.0 Turorial Download page
Downloading the image wright_flight.jpg and storing it somewhere convient on your system.

This example assumes that you have the setup of at the end of the first part of the tutorial. The Chimera Website should be running and set to the values from the previouse example. This can be verified by using your Chimera Server to verify that you have the following values:

  • Check that you are in bill_web by clicking on the Webs tab.
  • Check that you have the places_to_go link set as the Active Link by clicking on the Links tab.

Jimage should be running and should have the following:

  • Check that it is connected to Chimera. (The status bar should indicate it is connected to the Chimera Server, the web bill_web, and to the active link places_to_go. See Disconnecting and Reconnecting the Jimage Client to Chimera for more information about the status bar.)
  • bill.gif should be displayed, and it should have an anchor on it which is connected to the Actvie Link (places_to_go).
  • earthrise.gif should be displayed, and it should have an anchor on it which is connected to the Active Link (places_to_go).

You are now ready to make an N-ary Link.
