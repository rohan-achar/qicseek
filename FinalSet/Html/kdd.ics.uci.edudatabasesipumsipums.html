IPUMS Census Database

Abstract

This data set contains unweighted PUMS census data from the Los Angeles and Long Beach areas for the years 1970, 1980, and 1990. The coding schemes have been standardized (by the IPUMS project) to be consistent across years.

Information files:

  • description of the data
  • codebook
  • ipums.la.names Contains variables and value descriptions.

Data files:

  • ipums.la.97.gz Los Angeles - Long Beach 1970 large version (1.7M; 13.6M uncompressed).
  • ipums.la.98.gz Los Angeles - Long Beach 1980 large version (2.0M; 14.5M uncompressed).
  • ipums.la.99.gz Los Angeles - Long Beach 1990 large version (2.3M; 17.2M uncompressed).
  • s.ipums.la.97.gz Los Angeles - Long Beach 1970 small version (200K; 1.4M uncompressed).
  • s.ipums.la.98.gz Los Angeles - Long Beach 1980 small version (232K; 1.5M uncompressed).
  • s.ipums.la.99.gz Los Angeles - Long Beach 1990 small version (278K; 1.7M uncompressed).
