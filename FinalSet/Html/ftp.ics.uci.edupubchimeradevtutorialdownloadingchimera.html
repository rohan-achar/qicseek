Downloading Chimera

  Download Chimera 2.0 Beta from the following site:
Chimera 2.0 Software page
If you just want the Java byte-code file for Chimera 2.0b1, just download the link labeled:
Chimera Class Files

This links allows you to download the chimera.jar contains all the Java byte-code for Chimera 2.0 Beta. This is what you need to run Chimera. You may place it anywhere you like. A convenient spot would be a location you commonly place your Java classes for your other Java programs.

You must then set the chimera.jar file up to be included in the CLASSPATH of your Java interpretor.

Look at the following links to see how you can set up your CLASSPATH to include the path to the chimera.jar file you have just downloaded:

  • Setting CLASSPATH in Solaris to include chimera.jar  
  • Setting CLASSPATH in Windows NT to include chimera.jar  

Note: Later on you will need to download client applications which will need to be placed in your CLASSPATH as well.

You are now ready to start the Chimera website.
