Strawman Definitions

  • Event notification service

    • No globally consistent definition or shared model

  • Notifications

    • Events, once observed at sources, generate notifications, which are then delivered to sinks

  • Notification messaging

    • Need an application-layer analogue to datagrams

  • Notification management mechanisms

    • Mailing list subscriptions, newsgroup creation, multicast address allocation, publish-and-subscribe, chat spanning tree construction, event coalescing
