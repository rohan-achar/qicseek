Joining Attributes

  • Joining Attributes

    • Treats a pair of attributes as a single attribute
    • Domain attributes being joined
    • Example
        height: (tall, short)

        age: (young, old)
        age_height:(young_tall, young_short, old_tall, old_short)
    • Useful if there are correlation among attributes
    • Estimates of probabilities

  • Partition Selection Problem-

    • Set of independent {{age}, {weight}, {height}, {shoe-size}}
    • Attribute {{age, height}, {weight}}
