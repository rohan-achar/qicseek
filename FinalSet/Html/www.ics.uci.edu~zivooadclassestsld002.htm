What is an Object (1)

  • An object is informally defined as a tangible entity that exhibits some well-defined behavior
    • a tangible and/or visible thing
      • Trucks, buildings, computers, sensors, books, people
    • something that may be apprehended intellectually
    • something toward which thought or action is directed
      • Bank account, transaction, array of integers, binary tree
