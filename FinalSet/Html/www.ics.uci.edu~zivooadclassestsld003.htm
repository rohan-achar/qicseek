What is an Object (2)

  • An object
    • exists in time and space
    • has crisply defined conceptual boundaries
    • may be tangible or intangible
      • Person, program, bank account, sales order, linked list
    • Definition: An object
      • has state
      • exhibits some well-defined behavior
      • has a unique identity
    • The structure and behavior of similar objects are defined in their common class
      • An object is an instance of some class
      • We use the terms object and instance interchangeably
