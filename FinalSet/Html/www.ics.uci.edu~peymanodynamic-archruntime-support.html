Runtime Support for
Architectural Dynamism

Publicly available tools that facilitate the use of dynamic software architectures, either through analysis, code generation, or runtime support are listed on this page.

ArchShell
Runtime support for dynamically modifying C2-style architectures written using the Java-C2 class framework.

Polylith
The Polylith and Clipper systems available by request.

Software Architect's Assistant
The Software Architect's Assistant provides a visual environment for the design, development, and runtime manipulation of Regis distributed programs. Regis is a programming environment for constructing distributed programs.
