Spatio-Temporal Uncertainty Reasoning System (SATURN)

Welcome to the Web Page of the SATURN Project

The work on Saturn is being conducted under two separate tasks.
  • The AWARE Project : whose goal is to develop a spatio-temporal database to support pilot advisory and weather event notification for general aviation pilots.

  • The VGIS Project: whose goal is to support army exercise data within Saturn.
